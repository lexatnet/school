# Черепаха

Это библиотека позволяет создавать графические приложения.
И на первый взгляд простые концепции предоставляют мощьный инструментарий.

Представим что у нас есть абстракный робот-черепаха который подчиняется нашим указаниям
во ворту у этой черепахи зажато перо или кисть мы можем давать команды этому роботу
двигаться вперёд, назад, менять перо на перо другого размера, менять цвет чернил,
прижимать перо к поверхности по которой он движется или поднимать.
Порехность по которой движется робот это бесеонечная плоскость.
Можно представить что робочерепаха двигается по огромному листу бумаги,
а мы видим ограниченую его часть через окно нашей программы.

## Команды для черепашки

### Перемещение

forward(distance) | fd(distance) - движение вперёд на отрезок длинной

backward(distance) | bk(distance) | back(distance) - движение назад на отрезо длинной

right(angle) | rt(angle) - поворот направо на угол

left(angle) | lt(angle) - поворот налево на угол

goto(x, y=None) | setpos(x, y=None) | setposition(x, y=None) - перемещение в позицию

setx(x) - устанавливаем позицию по координате - X

sety(y) - устанавливаем позицию по координате - Y

setheading() | seth() - устанавливаем напрвление
| standard mode | logo mode   |
| 0 - восток    | 0 - север   |
| 90 - север    | 90 - восток |
| 180 - запад   | 180 - юг    |
| 270 - юг      | 270 - запад |

home() - перемещение черепахи в начало координат

circle(radius, extent=None, steps=None) - рисуем круг заданого радиуса
radius - радиус
extent - градус какая часть круга будет нарисована
steps - количество шагов используемое для рисования круга

dot(size=None, *color) - рисуем точку
size – размер точки. целое >= 1 (если задаётся)
color – имя цвета представленное строкой или кортеж определяющий цвет в схеме (RGB)

stamp() - ставим штамп или отпечаток соответспующий форме черепахи
в позиции в которой находится черепаха возвращает идентификатор штампа

clearstamp(stampid) - удаляет штамп с заданным идэнтификатором

clearstamps(n) - удаляем все или первые/последние n штампов
n - целое число
если n > 0 удаляем n первых штампов, иначе если n < 0 удаляем последние n штампов.

undo() - отменяем последнее действие, количество действий которые мы можем отменить
определяется размером буффера который сохраняет совершенные действия

speed() - устанавливаем скорость



### Узнаем состояние

position() | pos() - в какой позиции черепашка

towards() - узнаём угол между направлением черепашки и вектором от черепашки до точки с координатами 

xcor() - какая координата по оси

ycor() - какая координата по оси

heading() - узнаём ориентацию черепашки

distance() - определение расстояния от черепашки до точки с координатами
