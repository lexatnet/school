# Черепаха
---
самая свежая документация доступна на английском языке [тут](https://docs.python.org/3/library/turtle.html)

Это библиотека позволяет создавать графические приложения.
И на первый взгляд простые концепции предоставляют мощьный инструментарий.

Представим что у нас есть абстракный робот-черепаха который подчиняется нашим указаниям
во ворту у этой черепахи зажато перо или кисть мы можем давать команды этому роботу
двигаться вперёд, назад, менять перо на перо другого размера, менять цвет чернил,
прижимать перо к поверхности по которой он движется или поднимать.
Порехность по которой движется робот это бесеонечная плоскость.
Можно представить что робочерепаха двигается по огромному листу бумаги,
а мы видим ограниченую его часть через окно нашей программы.

### Внешний вид
---
```python
shape(name)
```
устанавливает или возвращает форму черепашки
формы доступные по умолчанию  “arrow”, “turtle”, “circle”, “square”, “triangle”, “classic”
также можно определить любую другую форму с помощью функции register_shape().

---
```python
resourced(rmode)
```
возвращает или устанавливает режим масштабирования черепахи
`rmode` – может принимать следующие значения “auto”, “user”, “noresize”
`auto`: адаптирует размер черепахи в зависимости от размера пера.
`user`: адаптирует размер черепахи в соответствии со значениями stretchfactor and outlinewidth (outline), которые устанавливаются функцией shapesize().
`noresize`: no adaption of the turtle’s appearance takes place.

если праметры не заданы возвращает размер экрана

```python

```

`llx` – число, x-координата левого нижнего угла холста
`lly` – число, y-координата левого нижнего угла холста
`urx` – число, x-координата правого верхнего угла холста
`ury` – число, y-координата правого верхнего угла холста

### Методы ввода


`default` - значение по умолчанию
